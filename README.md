# 동구리 오피스 🏢

AI 직원들이 일하는 픽셀 아트 가상 오피스.
지시를 내리면 **팀장이 업무를 분배하고, AI 직원들이 각자 실제 작업을 수행**합니다 — 말풍선과 애니메이션으로 일하는 모습을 구경하면서.

## 구성

| 버전 | 경로 | 실행 엔진 | 특징 |
|---|---|---|---|
| API 버전 | `동구리오피스.html` | Anthropic / OpenAI API | 파일 하나로 동작. 브라우저에서 바로 열기 |
| Claude Code 연동 버전 | `동구리오피스-서버/` | Claude Code CLI (구독) | **실제 파일 수정·git·코드 실행** 가능 |

## 동작 방식

```
사장(사용자) 지시
   │
   ▼
김팀장 — 질문이면 답변만, 작업이면 JSON 분배안 생성
   │
   ├─ 이대리(개발) ─┐
   ├─ 박주임(디자인) ├─ 병렬 작업 (직원별로 다른 모델 두뇌 선택 가능)
   ├─ 최과장(분석)   │
   └─ 정사원(검토) ─┘
   │
   ▼
김팀장 — 결과 취합 → 최종 보고서 + 프로젝트 컨텍스트 자동 갱신
```

## Claude Code 연동 버전 실행

```bash
cd 동구리오피스-서버
node server.js
# 브라우저에서 http://localhost:3777
```

- API 키 불필요 — 로컬에 설치된 Claude Code(구독)로 실행
- Codex CLI가 설치돼 있으면 직원 두뇌로 GPT/Codex도 선택 가능
- 실행 권한 3단계: 제안만(읽기 전용) / 파일 수정 허용 / 전권 위임
- 추론 수준(extended thinking), 완료 알림음, 결과물 로컬 저장

## 주요 기능

- 픽셀 아트 오피스: 걷기·앉아서 일하기 애니메이션, 모니터에 흐르는 코드, 말풍선
- 팀장 분배안 패널: 누가 뭘 맡았는지, 직원별 결과 전문, 팀장 원문 JSON까지 투명하게 표시
- 프로젝트 컨텍스트: 직원들이 항상 참고하는 프로젝트 정보 — 작업이 끝나면 팀장이 스스로 갱신
- 추천 지시: 작업 완료 시 팀장이 "다음에 할 일" 3개를 제안해 버튼으로 갱신

## 기술

바닐라 JS + Canvas (프레임워크·빌드 없음), Node.js 내장 http 서버, Claude Code headless mode (`claude -p`)
